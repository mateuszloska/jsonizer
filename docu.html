JSONizer 1.1. by Mateusz Loska

JSONizer application converts .csv files into .json files. Please give the name of the file you want to convert to json and the path you want to save the json data in.

python jsonizer.py INPUT_FILE.CSV INPUT_FILE.JSON [SEPARATOR] [filter] [FILTER_KEY] [FILTER_VALUE]

depending on which separator you use in the CSV file you can specify the separator the default value is comma, so you can leave the field empty you can either pass a string or use one of the predefined values: comma space tab semicolon
example:

python jsonizer.py file1.csv file2.json semicolon

you can also filter the data, so that only values matching the key and its value will be shown
example:

python jsonizer.py file1.csv file2.json semicolon filter surname kowalski (converts only rows that match surname value of "kowalski" to json) when we use comma as separator we could use the shorter version: python jsonizer.py file1.csv file2.json filter surname kowalski

Examples

Converting simple .csv data to json with the default comma separator


You can convert simple csv files that are saved with the comma separator as following:

python jsonizer.py dummy_data.csv dummy_data.json
[
{
"id":"1",
"surname":"Kowalski",
"country":"PL",
"age":"32",
"job":"programmer"
},
...

Converting a .csv file with tab separator


As .csv files could be saved with different separators you can use some predefined values such as tab , space or semicolon to convert the data.

python jsonizer.py dummy_data_tab.csv dummy_data_tab.json tab
[
{
"id":"1",
"surname":"Kowalski",
"country":"PL",
"age":"32",
"job":"programmer"
},
...

Filtering and converting a .csv file with custom separator


You can filter a .csv file and generate json data using filters. In this example you can see, how to filter out all the data with the country "UK" from the dummy data, so that only the rows containing for country will be converted to json.

python jsonizer.py dummy_data_letter.csv dummy_data_letter.json p filter country UK
[
{
"id":"5",
"surname":"Kij",
"age":"50",
"job":"athlete"
},

"age":"45",

},
